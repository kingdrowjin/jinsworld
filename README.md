# Let's Hang - Event Creation Page

A React-based event creation page that allows users to create and customize events with flyers, backgrounds, and customizable modules.

## Live Demo
Run locally: `npm run dev` → http://localhost:5174

## Features

- Flyer image upload with native file picker
- Background image change with gradient overlay
- Event details form (phone, date/time, location, cost, description)
- Customizable modules (Capacity, Photo Gallery, Links)
- Quick-link modules (RSVP, Schedule, Menu, Music, Dress Code, Registry)
- Go Live functionality

## Tech Stack

- React 19
- Vite
- Custom CSS with glassmorphism effects

## How to Run

```bash
npm install
npm run dev
```
